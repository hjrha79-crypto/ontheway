OnTheWay 프로젝트에서 다음 4가지를 수정해줘:

1. CallFilter.judge()에서 ACCEPT일 때 reason을 구체적으로 변경:
   - 거리 있을 때: "금액 N,NNN원, 거리 N.Nkm, 단가 N,NNN원/km ≥ 2,000원 기준 충족"
   - 거리 없을 때: "금액 N,NNN원 ≥ 최소기준 3,000원 충족"
   - REJECT일 때도: "단가 N,NNN원/km < 2,000원 기준 미달" 또는 "금액 N,NNN원 < 최소기준 3,000원 미달"

2. 고액 콜 보호: CallFilter.judge()에서 금액이 7,000원 이상이면 단가와 무관하게 ACCEPT 처리. reason: "고액 콜 N,NNN원 ≥ 7,000원 (단가 무관 통과)"

3. 콜 상세 다이얼로그에서 "가게: 배민배달", "목적지: 지도앱으로 검색하기" 같은 의미 없는 텍스트는 표시하지 않기. 가게명이 플랫폼명과 같거나, 목적지에 "검색하기"가 포함되면 해당 줄 숨기기.

4. DeliveryNotificationService.kt에서 Toast.makeText가 포함된 모든 줄을 Log.d로 완전히 교체. Handler(Looper.getMainLooper()).post { Toast... } 패턴도 포함. android.widget.Toast import 제거.
