OnTheWay 프로젝트에서 v3.3 추가 기능을 모두 구현해줘:

1. 즐겨찾기 가게 관리:
   - 상단 "즐겨찾기" 버튼 클릭 시 즐겨찾기 화면 표시
   - 콜 상세 다이얼로그에 "즐겨찾기 추가" 버튼 추가
   - 즐겨찾기 가게에서 콜이 오면 판정에 보너스 (기준 -1,000원 완화)
   - 반대로 "블랙리스트" 기능도 추가 — 블랙리스트 가게는 무조건 REJECT
   - reason에 "즐겨찾기 보너스" 또는 "블랙리스트 거부" 표시
   - TTS: 즐겨찾기 가게면 "단골 가게입니다" 추가 멘트
   - SharedPreferences에 JSON으로 저장

2. 피크 시간 감지:
   - 시간대별 콜 빈도를 분석하여 피크 타임 자동 감지
   - FilterLog에서 최근 7일간 시간대별 콜 수 집계
   - 피크 시간대(상위 20%)에는 기준 자동 상향 (+500원) — 콜이 많으니 더 좋은 것만 잡기
   - 비피크 시간대에는 기준 자동 하향 (-500원) — 콜이 적으니 기준 완화
   - 상태 탭에 "현재 피크 타임" 또는 "비피크 타임" 표시
   - 설정: "피크 자동 조절" 토글 (기본 OFF)

3. 연속 REJECT 경고:
   - 5건 연속 넘기세요가 나오면 TTS: "5건 연속 넘기고 있습니다. 기준을 낮춰볼까요?"
   - 10건 연속이면 TTS: "콜이 계속 안 맞습니다. 이동을 추천합니다."
   - 상태 탭에 "연속 넘김: N건" 카운터 표시
   - 설정: "연속 넘김 경고" 토글 (기본 ON)

4. 배달 완료 추적:
   - Accessibility로 "배달 완료", "배달완료", "배달이 완료되었습니다" 텍스트 감지
   - 감지되면 EarningsTracker에서 해당 건을 "완료"로 마킹
   - 완료 시 TTS: "배달 완료. 오늘 N건 완료, 매출 N원"
   - 수락→완료 시간차이로 건당 소요시간 자동 계산
   - 통계에 "평균 배달 소요시간" 추가
   - 설정: "배달 완료 안내" 토글 (기본 OFF)

5. 콜 알림 사운드:
   - TTS 외에 잡으세요/괜찮습니다/넘기세요별 다른 알림음 재생
   - 잡으세요: 밝고 긴급한 차임벨 (res/raw/sound_grab.mp3)
   - 괜찮습니다: 부드러운 알림음 (res/raw/sound_ok.mp3)
   - 넘기세요: 짧은 경고음 (res/raw/sound_skip.mp3)
   - 사운드 파일이 없으면 기본 시스템 알림음 사용
   - 설정: "콜 알림음" 토글 (기본 OFF), TTS 전/후 선택

6. 오늘의 목표 설정:
   - 설정에서 일일 목표 금액 설정 (기본 100,000원, 10,000원 단위)
   - 일일 목표 건수 설정 (기본 30건)
   - 목표 시급 설정 (기본 15,000원/h)
   - 목표 달성률 상단에 프로그레스바로 표시
   - 목표 50% 달성 시 TTS: "목표 절반 달성!"
   - 목표 100% 달성 시 TTS: "오늘 목표 달성! 수고하셨습니다!"
   - 설정: "목표 알림" 토글 (기본 OFF)

수정 후 git add -A && git commit -m "v3.3: 즐겨찾기가게 + 피크감지 + 연속REJECT경고 + 배달완료추적 + 알림사운드 + 목표설정" && git push 해줘.
